# Writer Original Specs

This folder preserves the upstream Writer Computer specs used as migration
source material for the Burrete transplant.

Do not treat these files as direct Burrete requirements by themselves. The
Burrete migration contract is defined by:

- `SPECs/full-writer-transplant-spec.md`
- `SPECs/writer-feature-port-map.md`

Each upstream Writer spec must be either copied into Burrete behavior, adapted
to molecular preview behavior, or explicitly marked deferred/disabled in the
feature port map.
